# Secure-Chat-App
Secure Chat App (PySide6 + RSA)
A beginner-friendly, modern GUI chat app featuring end-to-end encryption and message signature verification. Built with Python, PySide6, and RSA cryptography for secure client-server messaging.

# Features:
Real-time messaging with multiple clients
RSA encryption/decryption for message security
Digital signature verification
Contacts management and live key updates
Modern GUI with message bubbles and scrollable chat
Audio notification on message send

# Tech Stack: 
Python, PySide6, Cryptography, Socket Programming, Pygame
